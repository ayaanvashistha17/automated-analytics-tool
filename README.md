# Automated Analytics & Predictive Modeling Tool

## Overview
A Python-based automation tool that generates daily status reports with predictive analytics capabilities. The tool integrates VBA macros for seamless Excel data refreshing and Power BI synchronization, and implements linear regression for trend analysis and forecasting.

## Features
- Automated daily report generation
- Predictive analytics using linear regression
- Excel automation with VBA integration
- 40% improvement in operational efficiency
- Power BI synchronization capabilities

## Tech Stack
- Python (Pandas, Scikit-learn, Openpyxl)
- VBA Macros
- Excel Automation
- Predictive Modeling
- Power BI Integration
